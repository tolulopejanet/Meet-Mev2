# Meet-Mev2

This project has been migrated to a Vite + Tailwind CSS frontend.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open the app in your browser at the local Vite URL.

## Notes

- Static images are served from `public/images`.
- The previous Python/Poetry files were removed.
- The app is now a Vite-powered frontend using Tailwind CSS.
